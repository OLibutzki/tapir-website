Licenses

tapir is shipped with an Evaluation license which can be used by everyone. This enables you to start with tapir without requesting a license key.

Users can request a personal license free of charge for testing non-commercial software. This license is not limited by the number of Page Objects or Test Cases.

For testing commercial software we offer correspsonding licenses. In case the licenses do not fulfill all your needs, contact us.

tapir does not restrict the number of concurrent users or executions. You do not have to care about the number of developers and tapir tests can be executed in parallel without any limitation. Therefore, scaling (Cloud, Docker, VM) does not cause additional licensing costs.
